# Amobee

Amobee (now rebranded as Nexxen) is a global digital advertising platform offering REST APIs for managing cross-channel programmatic campaigns, audience targeting, data management, and advertising analytics. The platform enables advertisers and agencies to plan, activate, and measure media across display, video, mobile, social, and TV channels through a unified Demand Side Platform (DSP).

## API

The Amobee Campaign API is a REST API with base URL `https://services.amobee.com/campaign/v3/api`. Authentication uses OAuth2 client credentials flow — clients obtain a Bearer token from `https://services.amobee.com/accounts/v1/api/token` using a `client_id` and `client_secret` provisioned by Amobee. The API supports management of advertisers, insertion orders, line items, packages, creatives, and ads.

Swagger/OpenAPI documentation is available at `https://services.amobee.com/campaign/v3/doc/` but requires an authenticated Amobee account to access.
